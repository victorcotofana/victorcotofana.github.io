1. Descrierea proiectului

Ideea aplicatiei este de a concepe un magazin web de accesorii de imbracaminte pentru acoperit crestetul capului. Utilizatorul are posibilitatea de a proba accesoriile fara a necesita prezenta fizica folosind realitatea imbogatita (AR â€“ Augmented Reality). Pe langa lista de accesorii valabile in magazinul online, utilizatorului ii vor fi oferite o lista de recomandari: in functie de sezonul curent si in functie de preferinta utilizatorului privind culoarea accesoriilor.

2. Navigarea

Utilizatorul are acces la urmatoarele pagini:

  • Home Page
  • All Accessories Page
  • Item Page
  • Favorites Page
  • Cart Page
  • User Page

Toate paginile in afara de Item Page pot fi accesate din toolbar-ul aplicatiei. Item page, pagina individuala a fiecarui accesoriu, este accesibila din pagina Home Page, All Accessories sau Favorites Page, apasand pe imaginea accesoriului.

3. Home Page

Prima pagina cu care interactioneaza utilizatorul este pagina principala. Aici utilizatorul poate gasi mesajul de bun venit si ultimele 5 accesorii adaugate in magazinul virtual. In partea de sus a paginii se afla toolbar-ul existent in toata aplicatia.

4. All Accessories Page

In pagina de accesorii utilizatorul poate gasit toate accesoriile din magazin la ora actuala. In partea stanga are doua card-uri cu selectie: Categories si Filter By. Utilizatorul poate selecta ce categorie de accesorii vrea sa vada, iar apoi poate filtra aceste accesorii pe baza culorii, genului, sau ordonarea pretului. Daca doreste poate elimina toate filtrele selectate mai devreme. Pentru a accesa pagina fiecarui accesoriu utilizatorul apasa imaginea accesoriului. Langa aceasta imagine exista si un buton in forma de inima. Daca este apasat accesoriul este adaugat in lista de favorite a utilizatorului.

5. Item Page

Fiecare accesoriu isi are propria pagina, unde poate fi gasit pretul si descrierea item-ului. La fel este prezent si butonul de adaugare la favorite. Sub imaginea accesoriului se afla doua butoane: Try it LIVE si Add to cart. Add to cart adauga produsul in cosul virtual al utilizatorului. Try it LIVE porneste camera Web al utilizatorului unde acesta poate proba accesoriul virtual. In timp ce probeaza accesoriul, utilizatorul isi poate face o poza. Toate pozele facute de utilizator sunt salvate local, si pot fi accesate in pagina utilizatorului (User Page). In partea de jos a paginii se afla recomandarile pentru utilizator. Aceseta sunt randate in dependenta de preferinta utilizatorului pentru culorile accesoriilor si sezonul curent.

6. Favorites Page

Dupa ce utilizatorul apasa pe butonul in forma de inima a unui accesoriu gasit pe parcursul aplicaitiei, toate accesoriile favorite vor fi gasite aici. Utilizatorul poate accesa pagina individuala a accesoriilor apasand pe imaginea lor respectiva, sau poate elimina accesoriul din pagina de favorite apasand din nou pe butonul in forma de inima.

7. Cart Page

Apasand pe butonul Add to cart utilizatorul este redirectionat la pagina cosului virtual. Aici isi poate vedea toate accesoriile din cosul sau. Utilizatorul poate elimina un accesoriu din cos, sau poate simula Check Out-ul.

8. User Page

In pagina utilizatorului pot fi gasite setarile pentru preferinta culorii accessoriilor si imaginile salvate de utilizator. Imaginile pot fi sterse sau descarcate direct de aici.
